## Lessons Learned

3. **Protocol extends beyond group chat** — Iris's protocol only said "group chat" but she named Pixel in a DM. The rule should apply to ALL outputs, not just group chat.

4. **Artemis is prone to timeouts** — Her MEMORY.md and large context may cause slower responses. May need timeout increase or model optimization.

5. **OpenClaw agent CLI works well** — `openclaw agent --agent <name> --message "..."` is fast and reliable for direct agent testing.

6. **Agents correctly check workspace first** — Shield checked shared workspace files before asking Fiez. This validates the protocol design.
