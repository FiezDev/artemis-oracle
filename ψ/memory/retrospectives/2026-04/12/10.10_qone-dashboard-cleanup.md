## Lessons Learned

1. **Husky + npm test with no tests = every commit fails silently** — The default `create-react-app` / `npm init` test script is `echo "Error" && exit 1`. Always check this when setting up husky.
2. **CamelCase vs snake_case mismatch between frontend and API** — Frontend sends `taskId`, API expects `task_id`. This is invisible until you trace the actual HTTP calls. Standardize one convention or add a mapping layer.
3. **Auth middleware must include dashboard itself** — When building a dashboard that writes on behalf of agents, the dashboard needs its own identity in the auth system.
4. **TypeScript union types are useless if interfaces use `string`** — Defining `TaskStatus` but then writing `status: string` defeats the purpose. Always use the union type in the interface.
