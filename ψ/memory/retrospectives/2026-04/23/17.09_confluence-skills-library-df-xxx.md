## Lessons Learned

- Atlassian MCP page IDs are only known after creation — main page links must be updated in a second pass with actual IDs
- Confluence `updateConfluencePage` rejects `parentId` when updating a page to point to itself
- `zip` not installed on this WSL2 — Python `zipfile` module works as fallback
- Evals contain local paths (e.g., `/home/bjgdr/dev-work/...`) and should be excluded from shareable zips
- Each skill child page follows consistent structure: command, triggers, what it does, phases/workflow, setup, usage examples
