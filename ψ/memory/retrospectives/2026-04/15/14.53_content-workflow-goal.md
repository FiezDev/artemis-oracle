## Lessons Learned

1. **Verify tool locations before planning** — Initial assumption that Z-Image was on Higgsfield caused rework. Always verify where tools/models actually live before building plans around them.
2. **Start QOne API before API calls** — The dashboard API needs to be running (`cd dashboard/api && bun run src/index.ts`) before any curl commands work. No auto-start.
3. **Task creation requires sprintId** — Cannot create tasks without a valid sprint reference. Must create sprints first, then tasks.
4. **HTML docs as plan artifacts work well** — Linked HTML pages with dark theme are readable, navigable, and don't require a build system. Good for spec deliverables.
