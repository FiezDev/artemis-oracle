## Lessons Learned

1. **API response shapes must match frontend expectations** — The stats endpoint returns objects not arrays. Always check both sides before assuming format.
2. **Docker fresh DB loses everything** — When migrating from local postgres to docker postgres, all data is gone. Need re-seeding scripts ready.
3. **zsh doesn't support bash associative arrays** — `declare -A` fails in zsh. Use simple curl loops or bash explicitly.
4. **Sprint route missing list endpoint** — Service had `listSprints()` but route never exposed it. Always verify route↔service alignment.
5. **Direct DB inserts for agents** — No agent registration API exists; agents are only created via heartbeat which requires them to exist first. Seed via SQL.
