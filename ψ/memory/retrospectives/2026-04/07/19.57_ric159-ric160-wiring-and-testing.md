## Lessons Learned

1. **Disconnected code is invisible code** — Prior session created domain files (models, queries, resolvers) but never registered them in GraphQL schema, DB schema, or app startup. The code existed but was unreachable. Always wire into the entry points.

2. **Zod transform without refine is a footgun** — `z.union([z.number(), z.string()]).transform(Number)` silently produces NaN for invalid strings. Always pair Number transforms with `.refine(v => !Number.isNaN(v))`.

3. **Test env bleeding** — TmdClient cached `TMD_BEARER_TOKEN` from `.env` at module load time. Tests that needed "no token" couldn't override it. Fix: accept options in constructor for testability.

4. **E2e tests shouldn't need a server** — Auth tests that hit `localhost:8080` are fragile. Direct resolver calls with mock context are faster, more reliable, and test the same logic.

5. **Context limits break continuity** — The prior session ran out of context mid-task. Session recovery summary was critical for knowing what was done vs what still needed doing. The plan file at `/tmp/clother-claude-config-161707806/plans/noble-enchanting-mountain.md` was the single source of truth.
