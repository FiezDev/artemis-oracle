## Lessons Learned

- **Agent-runners short-circuit in this dashboard** — they "complete" wakeups via openclaw adapter without actually executing the work (channel-watch returns "no new videos" while gate items contain rich content; gate auto-approves as `artemis` without human input; reel agent claims success without calling NanoBanana). Direct-bypass scripts are the reliable path until the openclaw tool config gets locked down.
- **Codex CLI failures are mid-execution, not rate-limit** — failures cluster at 88-113s elapsed, not 0-5s. Means short backoff (5s) is sufficient because there's no cooldown window to wait through; longer backoffs just waste time. Successful gen has median 137s.
- **`/tmp` gets garbage-collected on this machine** — both `/tmp/aisearch5/` (variant JSONs, helper scripts) and `/tmp/page-token.txt` disappeared between sessions. Persistent state belongs in `~/.fb-page-token-aiinspire` (chmod 600) or vault, not `/tmp`.
- **Graph Explorer tokens default to short-lived** — must use Access Token Debugger → "Extend Access Token" to get a 60-day user token before deriving the page token from `/me/accounts`. Page tokens derived from long-lived user tokens inherit non-expiry.
- **REASONABLE STORY BOUNDARIES > fixed topic counts** — earlier templates that said "expect 3-6 topics for roundups, 1-2 for deep dives" produced under-coverage of news roundups. Replacing with "same story = one topic, different story = different topic, judged from natural transcript seams" gave 18 stories per news roundup vs 5 previously.
- **Video host (AI Search) is NOT authoritative** — ~30% of fact-checked claims needed correction (impossible numbers, mistranscribed product names, vendor misattribution). Web research must be a mandatory step, not optional. Encoded as red lines in the strict fact-check template.
- **Per-node `variable_overrides` UI is the right escape hatch** — letting one process node be configured per workflow (e.g., AI Daily with `videoCount=5` and Long Form with `videoCount=1`) means the same process slug can serve different cadences without forking the template. Used heavily this session for drip + retry config.
